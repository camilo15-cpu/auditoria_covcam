# Registro Transversal de Prompts de IA

Bitácora de interacciones con el asistente de Inteligencia Artificial para el desarrollo de la auditoría:

1. *Diseño de Entorno:* Solicitudes para estructurar el proyecto en React estructurando de forma modular los componentes del Portal de Auditoría.
2. *Generación de payloads:* Consultas de asistencia técnica para emular los vectores de ataque (SQLi, XSS, Command Injection) bajo entornos controlados de laboratorio (DVWA).
3. *Modelamiento de Negocio:* Estructuración de los impactos de ciberseguridad adaptados al ecosistema municipal de la Comuna de Cerro Verde.
