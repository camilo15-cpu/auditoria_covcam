# Identificación de Activos de Información - Cerro Verde

Para la Municipalidad de Cerro Verde se han definido los siguientes activos de información críticos asociados al Portal de Trámites:

1. **BD-VECINOS (Base de Datos de Usuarios):** Contiene RUTs, nombres, direcciones y correos. (Criticidad: Alta)
2. **SYS-RECAUDA (Sistema de Pagos y Patentes):** Motor de software que procesa las transacciones financieras y comerciales. (Criticidad: Crítica)
3. **SERV-WEB (Servidor de Aplicación Linux):** Infraestructura subyacente montada sobre el sistema operativo que aloja la web institucional. (Criticidad: Crítica)
